#  Travel Bucket List Agent — TSU

A conversational travel planner built with LangGraph, LangChain, and OpenAI GPT-4o-mini.

---

## Features
- Add/list/remove travel destinations
- Country facts via REST Countries API
- Live weather via Open-Meteo API
- Token usage tracking
- Human approval before deleting destinations

---

## Setup
1. Open `AGENT1.ipynb` in Google Colab
2. Run Cell 2 (pip install)
3. Paste your OpenAI API key in Cell 3
4. Run all cells top to bottom

---

## Requirements
- Python 3.10+
- OpenAI API key (gpt-4o-mini)
- Google Colab (free)
- Internet connection

## Dependencies
langgraph, langchain, langchain-openai, requests, pydantic
